### 371. Sum of Two Integers

:orange_circle: Medium

Given two integers `a` and `b`, return _the sum of the two integers without
using the operators `+` and `-`._

__Example 1:__
<pre>
<b>Input:</b> a = 1, b = 2
<b>Output:</b> 3
</pre>

__Example 2:__
<pre>
<b>Input:</b> a = 2, b = 3
<b>Output:</b> 5
</pre>

__Constraints:__

* `-1000 <= a, b <= 1000`
